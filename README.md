*Simple API LINE Notify*

----

## Requirement

* Node.js v8.11.4
* NPM v6.4.0
* request
* request-promise
* fs

## Installation

Installation is simple. It can be installed from NPM from github:
```sh
$ npm install https://github.com/albertoanggi/js-line-notify
```
